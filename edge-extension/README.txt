# CALCLiK Microsoft Edge Extension

AI-powered event scanner for Microsoft Edge browser.

## Installation

### Option 1: Developer Mode (Recommended for Testing)
1. Open Microsoft Edge
2. Navigate to `edge://extensions/`
3. Enable "Developer mode" toggle in the left sidebar
4. Click "Load unpacked"
5. Select this directory (edge-extension)
6. The extension icon will appear in your toolbar

### Option 2: Edge Add-ons Store (Coming Soon)
The extension will be available on the Microsoft Edge Add-ons store once published.

## Features

- AI-powered event detection on any webpage
- Local processing using Transformers.js (no API key required)
- One-click calendar export (iCal format)
- Support for multiple calendar platforms
- Privacy-first design - all processing happens in your browser

## Usage

1. Navigate to any webpage with events
2. Click the CALCLiK extension icon
3. Click "Scan Page" to detect events
4. Review detected events
5. Click "Add to Calendar" to export to your preferred calendar app

## Supported Platforms

- Microsoft Outlook Calendar
- Google Calendar
- Apple Calendar/iCal
- Any calendar app that supports .ics files
