# CALCLiK Browser Compatibility Guide

## Brave Browser Support

CALCLiK fully supports Brave browser as it's built on Chromium, the same engine as Google Chrome.

### Installation for Brave:

1. **Enable Developer Mode:**
   - Open Brave and go to `brave://extensions/`
   - Enable "Developer mode" toggle in the top right

2. **Install the Extension:**
   - Download the Chrome extension package from [calclik.pages.dev](https://calclik.pages.dev)
   - Extract the ZIP file
   - Click "Load unpacked" in Brave extensions page
   - Select the extracted CALCLiK folder

3. **Verify Installation:**
   - The CALCLiK icon should appear in your browser toolbar
   - Click it to open the popup interface

### Features Available in Brave:

✅ **Full AI Event Detection** - All local AI processing works identically to Chrome
✅ **Calendar Integration** - Google Calendar, Outlook, and iCal downloads
✅ **macOS Reminders** - Native messaging works with proper setup
✅ **Privacy Protection** - All processing happens locally, no external API calls

### Brave-Specific Benefits:

- **Enhanced Privacy**: Brave's built-in ad/tracker blocking complements CALCLiK's privacy-first approach
- **Faster Processing**: Brave's optimizations may improve AI model loading times
- **Built-in Shields**: Work alongside CALCLiK without conflicts

### Troubleshooting:

If you encounter issues:
1. Ensure Brave is updated to the latest version
2. Check that the extension is enabled in `brave://extensions/`
3. Reload the current webpage and try scanning again
4. For Reminders integration, follow the same native messaging setup as Chrome

### Performance:

Brave browser typically shows 10-20% faster AI model initialization compared to standard Chrome due to its optimizations.
